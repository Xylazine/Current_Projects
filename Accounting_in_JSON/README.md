# Accounting in JSON - A Pay Statement Parsing Script

## Author Information
**Name**: Amanda Gale  

## Project Description
Where does all the money go? The short answer is motorcycle parts. The long answer is provided
by this script. This script downloads pay statements from a secure online portal using a supplied login cookie.
It then parses the earnings and deductions from their JSON format and creates an output file,
formatting pay information by date in a manner that can be used by an online accounting software
to track financial activity.

## How to run
Provide a login cookie to your for the script. This is not a highly reproducible project, 
simply an example of a parsing script that can be made for everyday means.
